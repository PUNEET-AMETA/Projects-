Study Assistant
Description: A Raspberry Pi-based system designed to monitor students' study sessions and promote healthy study habits by addressing distractions, posture issues, and hydration reminders.
Technologies Used: Raspberry Pi, Python, PiCamera, 20x4 LCD, buttons, and speakers.
Key Features:
Detects face and eye status to alert when the user is inattentive or asleep.
Monitors posture and provides reminders to correct unhealthy positions.
Tracks session duration and logs data into an Excel file.
Includes a water break and sedentary break reminder system.
Displays real-time information on a 20x4 LCD screen.
Achievements: Successfully implemented live eye tracking and data logging, improving user focus and reducing distractions during study sessions.
